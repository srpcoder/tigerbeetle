# `lookup_transfers`

Fetch one or more transfers by their `id`s.

## Event

An [`id`](../transfers.md#id) belonging to a [`Transfer`](../transfers.md).

## Result

- If the transfer exists, return the [`Transfer`](../transfers.md).
- If the transfer does not exist, return nothing.
